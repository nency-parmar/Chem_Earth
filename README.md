### Prerequisites
- Flutter SDK (3.5.0 or higher)
- Dart SDK
- iOS Simulator / Android Emulator / Physical Device

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd ChemEarth
   ```

2. **Install dependencies**
   ```bash

   ```

3. **Run the application**
